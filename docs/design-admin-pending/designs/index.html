위장 강화 원칙

모든 화면에서 "chat / secret / agent / message" 어휘 노출 0. 외부 관찰자에게는 끝까지 평범한 뉴스 앱.

채팅 화면 위장 · ArticleDetailScreen 내부 conditional swap. 헤더 타이틀 = "기사 토론", placeholder = "댓글 입력", 상단 핀 = "토론 중인 기사".
알림 위장 · notification payload만, data 0. 표시 = 앱 이름 · 헤드라인 한 줄 · 매체 · 시각. 발신자 / 미리보기 미노출.
설정 어휘 위장 · 시퀀스 = 콘텐츠 잠금 코드, 페어 = 구독자, reset = 잠금 해제 도움말.
App Switcher 위장 · 채팅 모드에서 백그라운드 전환 시 일반 피드로 즉시 swap. 스크린샷 차단 (Android FLAG_SECURE, iOS blur).
