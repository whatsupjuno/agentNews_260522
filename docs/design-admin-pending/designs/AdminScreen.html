DailyNews MVP 운영용 모바일 관리자 앱. 진입은 사용자 앱의 ARM 시퀀스 로만 가능 — 7·1·3·5 (사회 → 헤드라인 → 경제 → 문화) 입력 시 AdminScreen.html 으로 이동.

5 화면 / 4 탭
01
홈 (Overview)
통계 카드 3개 (전체 사용자 / 활성 채팅방 / 누적 메시지) + 3 메뉴 진입점. MVP 운영 정책 안내 박스.
02
사용자 관리
검색 + 4-segment 필터 (전체/활성/미인증/차단). 행 탭 → 상세. 활성 3 · 미인증 2 · 차단 1.
03
사용자 상세
프로필 카드 + 계정 정보 + 페어 연결 + 잠금 코드 (편집 진입) + 위험 작업 (차단 / 삭제). @jun_2026 예시.
04
ARM 관리
현재 시퀀스 카드 4개 + "시퀀스 편집" 인터랙티브 + 카테고리 위치 매핑. 5·3·1·7 변경 시 모든 unlock_token 무효화.
05
채팅 데이터
통계 3개 + 채팅방 리스트 (페어 아바타 / UUID / 용량 / 마지막 활동). 행 탭 → 인라인 상세 + 메시지 내보내기 / 삭제. 일괄 작업.

진입 메커니즘

사용자 앱의 NewsFeedScreen 에서 DailyNews 워드마크 탭 → ARM (8초) → 카테고리 pill 7·1·3·5 시퀀스 → localStorage.dn_admin = "1" 세팅 후 AdminScreen.html 로 이동.

RN production: navigation.navigate('AdminStack') 로 별도 네비게이션 스택 진입.

채팅 시퀀스와 분리

일반 사용자 시퀀스 = 5·3·1·7 (문화·경제·헤드라인·사회) → 채팅 unlock.
Admin 시퀀스 = 7·1·3·5 (역순) → Admin 콘솔.
